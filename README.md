# belajarcrud-php
Belajar CRUUD PHP


Cara Menggunakan :
1. Download repository belajarcrud-php
2. Ekstrak file nya
3. Pindahkan ke dalam folder htdocs
4. Eksport database
5. Cocokan nama database di file function
6. Selesai.
